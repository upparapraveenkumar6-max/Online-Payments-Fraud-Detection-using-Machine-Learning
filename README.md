# Online-Payments-Fraud-Detection-using-Machine-Learning
This project is a Machine Learning based web application that detects fraudulent online payment transactions. The system analyzes transaction details and predicts whether a transaction is Fraudulent or Not Fraudulent.

## Technologies Used
- Python
- Machine Learning
- Flask
- Bootstrap

## How to Run the Project

1. Clone the repository
2. Install required libraries
3. Run the Flask app

python app.py
